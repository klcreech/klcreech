### 🛠️ Current Focus: Building the "Agentic Edge"
I don't just "wrap" APIs. I build systems where AI interacts with reality. 
* **Discovery:** Agents that "sense" their OS, hardware, and network state before acting.
* **Orchestration:** Creating pipelines that chain local LLM reasoning with native system commands.
* **Safety:** Environment-aware logic that distinguishes between restricted containers and host-level access.
